## Java 7강
#### 배열 : 인덱스를 이용해 자료형이 같은 데이터를 관리하는 것
#### 배열 선언과 초기화 과정은 다음과 같다
``` java
int [] arr1 = new int[5];
arr1[0] = 10;
arr1[1] = 20;
arr1[2] = 30;
arr1[3] = 40;
arr1[4] = 50;
```
``` java
int [] arr1 = {10,20,30,40,50};
```
