## Javascript 강의 2-14
#### if와 else 쓰기
``` java
if (조건문) {
    동작문;
} else {
    동작문;
}
```
#### else if : 파이썬의 elif와 달리 상위 조건문 충족 시 아래 조건문 고려 x
#### 막연하게 들여쓰기를 하다보면 콜백지옥에 빠질 수 있음
#### switch 조건문 : 조건식과 비교조건식이 같을 때 동작
``` java
switch (조건식) {
    case 비교조건식 :
      동작문;
}
```
#### switch는 하나의 조건식이 만족하더라도 아래에 조건 모두 실행, 이걸 막으려면 break
#### default 는 switch 조건문에서 else와 같은 역할
